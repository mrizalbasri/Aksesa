# Aksesa - Backend Services

## Azure Integration Modules

### azure_openai.py
Module untuk integrasi Azure OpenAI GPT-4o
- Generate rekomendasi personal
- Analisis narasi otomatis
- Explainable AI explanations

### azure_ml.py
Module untuk integrasi Azure Machine Learning
- Credit scoring model inference
- Model management
- Batch predictions

### azure_docintel.py
Module untuk integrasi Azure Document Intelligence
- OCR untuk invoice/nota
- Data extraction dari dokumen
- Quality validation
